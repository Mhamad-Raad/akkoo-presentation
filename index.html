Akkooo Orders

A Modern, Real-Time Order Management Dashboard

Project Description

Akkooo Orders is a modern, real-time dashboard for managing and tracking customer orders across multiple stages. Built using the latest web technologies, it supports advanced filtering, sorting, drag-and-drop status changes, theme switching (dark/light), and bulk operations. The dashboard simulates real-time updates via Firebase Firestore, providing an interactive and responsive experience ideal for managing live order workflows.

Features

  • Real-time order simulation and status updates
  • Drag-and-drop status management
  • Advanced filtering and search
  • Sorting by date, amount, or customer
  • CSV export and bulk actions
  • Dark and light theme support
  • Responsive and mobile-friendly UI

Technologies and Techniques Used

  • React 19 + TypeScript: Core framework for building UI with type safety.
  • Redux Toolkit + React-Redux: Efficient global state management with minimal boilerplate.
  • Firebase Firestore: Real-time cloud database for syncing orders.
  • Material UI (MUI): UI component library with full theme support (dark/light mode).
  • Dnd-kit: Powerful drag-and-drop library for managing orders between statuses.
  • Dayjs + @date-io/dayjs: Lightweight date utilities for filtering and range controls.
  • Notistack: Snackbar notifications for status changes and updates.
  • Vite: Lightning-fast development/build tool optimized for modern React.
  • ESLint + TypeScript ESLint: Ensures consistent, error-free code quality across the app.
  • MUI X Date Pickers: Rich date input components for custom date ranges.
  • Emotion: CSS-in-JS styling solution used under the hood by MUI for dynamic styles.
